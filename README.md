## Netflix Clone Projesi

Bu proje ile Netflix clone sayfası oluşturulmuş ve film verilerinin alınarak ekrana bastırılması amaçlanmıştır.

## Kullanılan Teknolojiler

-redux-thunk
-react-router-dom
-redux
-react-redux
-axios
-tailwind
-react-splide
-millify
